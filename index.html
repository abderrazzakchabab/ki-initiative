Schulungspaket DevOps (2x2 Tage) 

Die DevOps Schulung ist die welche am tiefsten in die Thematik Elasticsearch eintaucht und Wissen vermittelt welches zum Betrieb einer Elastic Infrastruktur benötigt wird. Dieser Kurs ist grundsätzlich für Mitarbeiter aus B&S vorgesehen welche sich Intensiv mit 
dem Betrieb von Elastic Search Infrastrukturen beschäftigen. Damit dieser Kurs den vollen Mehrwert entfalten kann, wird ein gewisses Basiswissen über Elasticsearch vorausgesetzt. 

Kurs Inhalt

Detaillierte Schulung zu den Themen

  • Elastic Search

  • Kibana (Fokus liegt auf Deployment und Betrieb von Kibana) 

  • Stack Features

  • Logstash

  • Beats / Agent / Fleet

Schulungspaket für Architekten (1 Tag)

Dieses Schulungspaket wurde so aufgebaut, dass in einem Tag eine gute Übersicht über die Produkte von Elasticsearch gegeben wird. Es wird auch eingegangen darauf wie der Elasticsearch Service des ISC aufgebaut ist und wie er am Besten in Anwendungen implementiert wird. 
Dieser Kurs ist für Architekten gedacht, damit diese in Zukunft in Projekten ein gutes Vorwissen haben wie Elasticsearch am Besten in Anwendungen integriert werden kann. 

Kurs Inhalt

Schulung zu den Themen

  • Elastic Stack

  • Elastic Cloud Enterprise 

  • Elastic Cloud Services

  • Elastic Cloud Kubernetes

  • Funktionsübersicht Elastic Stack und ESS Service im ISC

Schulungspaket für Entwickler und Anwender (3 Tage)

Diese Schulungspaket ist so aufgebaut, dass den Entwicklern und Anwendern eine gute Grundlage für den Umgang mit Elasticsearch und Kibana vermittelt wird. Es wird darauf eingegangen was der Elastic Stack bietet und wie damit gearbeitet wird. 
Der Fokus dieser Schulung liegt ausserdem auf dem Umgang mit Kibana. die Teilnehmer erfahren detailliert wie Kibana funktioniert, wie Index und Dashboards gebaut aufgebaut werden usw. 

Kurs Inhalt

Detaillierte Schulung zu den Themen

  • Übersicht Elastic und Cloud Dienste 

  • Elastic Search

  • Kibana

  • Stack Features (Reporting, Graph, Machine Learning, Canvas)
